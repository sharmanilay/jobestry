# Credits and Acknowledgments

Jobestry is built on top of excellent open-source projects and tools. We're grateful to all the contributors and maintainers who made this project possible.

## Core Framework

### Chrome Extension Boilerplate React Vite

Jobestry is built using the [Chrome Extension Boilerplate React Vite](https://github.com/Jonghakseo/chrome-extension-boilerplate-react-vite) by [Jonghakseo](https://github.com/Jonghakseo).

This boilerplate provides:
- React + TypeScript setup
- Vite build system
- Turborepo monorepo structure
- Hot Module Reload (HMR)
- Development tooling and configurations

**License**: MIT  
**Repository**: https://github.com/Jonghakseo/chrome-extension-boilerplate-react-vite
